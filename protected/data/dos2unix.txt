Dos2Unix / Unix2Dos - Text file format converters
http://waterlan.home.xs4all.nl/dos2unix.html

dos2unix and unix2dos utilities are used on Windows systems
(i.e. an EPD2 development workstation) so that CSV and text
files checked out on a Windows system have the same EOL marker
as when checked out on Unix (production) system.

Features (from Dos2Unix project web site)
* Native language support: Dutch, English, Esperanto, German, Spanish.
* Automatically skips binary and non-regular files
* In-place, paired, or stdio mode conversion.
* Keep original file dates option.
* 7-bit and iso conversion modes like SunOS dos2unix.
* Conversion of Windows Unicode (UTF-16) files to Unix Unicode (UTF-8)
* Handles Unicode Byte Order Mark (BOM)
* Secure

Distribution license, FreeBSD (GPL compatible)
