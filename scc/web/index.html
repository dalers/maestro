Maestro Users (OpenLdapAdmin)

User login id's, passwords, and email addresses are stored in an LDAP (light directory access protocol) directory. You can view all data anonymously (except for passwords), or login to modify your own data (such as your Maestro password). You cannot modify data of others users.

Changing your password this way is arguably not easy, but it works for now and an improved interface is in the works.

  • User login: "cn=lname\2C fname,ou=People,ou=scc,dc=root,dc=org", password: "appleton" (unless changed)
  • Admin login: "cn=Manager,dc=root,dc=org", password: appleton
