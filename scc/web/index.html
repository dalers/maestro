Maestro Users

User login id's, passwords, and email addresses are stored in an LDAP (light directory access protocol) directory. You can view all data anonymously (except for passwords), or login to modify your own data (such as your Maestro password). You cannot modify data of others users.

Changing your password this way is arguably not easy, but it works for now (a simplified interface to modify your password may be included in MaestroMail).
